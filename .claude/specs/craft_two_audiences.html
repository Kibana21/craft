Key differences
Same app. Same login URL. Different experience. When an internal staff member logs in with their AIA corporate credentials, they see CRAFT in full creator mode — projects, brand library management, compliance controls, analytics. When an FSC logs in with their agent credentials, they see CRAFT in agent mode — simpler, faster, focused on create and publish. The role is detected at login. No switching needed.

↑ Upstream — what flows back to internal
Usage analytics
Brand team sees which library assets FSCs remix most. Informs which content to create more of.
Compliance visibility
Every FSC export is logged with compliance score. Brand/compliance team can see all exports across the org — no surprises.
Content gap signals
If FSCs are creating a content type that's NOT in the library — that's a gap. Brand team gets notified to create the official version.
Top performing content
Which taglines, layouts, and themes get remixed most. Brand team learns what works for FSCs without running surveys.
